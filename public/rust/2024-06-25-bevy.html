01介绍

一个游戏框架 设计模式是ECS 面向数据编程

02组成

1、Apps

一个app组成主要通过 world, schedule, and runner 三部分组成

world: 世界 存储我们需要的游戏数据

schedule: 调度 调度相关的系统 来操作数据

runner：运行程序 解释调度

2、ECS

Entity Component System

一种程序设计模式 面向数据 使用bevy 开发游戏 需要遵循该模式 这种模式可以和面向对象oop对比

Entities : 实体 （一个具体的人 一条具体的狗）

struct Entity(u64);

Component : 组件 属性、抽象概念 (人有名字 狗也有)

#[derive(Component)]
struct Position { x: f32, y: f32 }

System: 控制实体运行 修改实体状态 添加实例 （实际就是一个函数）

fn print_position_system(query: Query<&Position>) {
    for position in &query {
        println!("position: {} {}", position.x, position.y);
    }
}

使用：


// Commands传递一些实例到世界

fn add_people(mut commands: Commands) {
    commands.spawn((Person, Name("Elaina Proctor".to_string())));
    commands.spawn((Person, Name("Renzo Hume".to_string())));
    commands.spawn((Person, Name("Zayna Nieves".to_string())));
}

fn greet_people(query: Query<&Name, With<Person>>) {
    for name in &query {
        println!("hello {}!", name.0);
    }
}
fn update_people(mut query: Query<&mut Name, With<Person>>) {
 // 通用调用时可带参数 query ,从 world拿包含Name的组件的实例，这些实例需要有Person属性

    for mut name in &mut query {
        if name.0 == "Elaina Proctor" {
            name.0 = "Elaina Hume".to_string();
            break; // We don’t need to change any other names
        }
    }
}

fn main() {

 // 给开始调度Startup添加系统 给loop调度Update 添加hello_world 
 // (update_people, greet_people)系统 chain使用串联 起他并行
    App::new()
        .add_systems(Startup, add_people)
        .add_systems(Update, (hello_world, (update_people, greet_people).chain()))
        .run();
}

3、Plugins

游戏模块化

impl Plugin for HelloPlugin {
    fn build(&self, app: &mut App) {
        app.add_systems(Startup, add_people)
            .add_systems(Update, (hello_world, greet_people));
    }
}


fn main() {
    App::new()
        .add_plugins((DefaultPlugins, HelloPlugin))
        .run();
}

4、Resources

基本都能用 Entity Component 来表达数据 但还有写全局唯一标识的数据

比如:

  • 运行时间
  • 资产集合(声音，纹理，网格)
  • 渲染器
#[derive(Resource)]
struct GreetTimer(Timer);

fn greet_people(
    time: Res<Time>,
    mut timer: ResMut<GreetTimer>,
    query: Query<&Name, With<Person>>
) {
    // time.delta()时间资源每次刷新的间隔
    // tick前进计时器 just_finished如果到了顶 返回true
    if timer.0.tick(time.delta()).just_finished() {
        for name in &query {
            println!("hello {}!", name.0);
        }
    }
}


impl Plugin for HelloPlugin {
    fn build(&self, app: &mut App) {
     // 添加资源 
        app.insert_resource(GreetTimer(Timer::from_seconds(2.0, TimerMode::Repeating)))
            .add_systems(Startup, add_people)
            .add_systems(Update, greet_people);
    }
}
