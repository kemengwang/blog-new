1. 移动零

lc

给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。

请注意 ，必须在不复制数组的情况下原地对数组进行操作。

思考：首先这个题要保证相对顺序，然后不能复制数组，这里显然需要原地交换，采用快慢双指针

快慢双指针：两个指针，初始在同一位置，然后向相同方向移动，一个移动速度快，一个移动速度慢。

适用场景：查找链表中间节点、链表是否包含环、原地修改数组。

快指针：寻找新数组的元素（非0元素），新数组就是不含有目标元素的数组 慢指针：指向更新 新数组下标的位置

/**
 * @param {number[]} nums
 * @return {void} Do not return anything, modify nums in-place instead.
 */
var moveZeroes = function(nums) {
    let slowIndex = 0
    for(let fastIndex = 0; fastIndex<nums.length; fastIndex++){
        if(nums[fastIndex] !== 0){
            [nums[slowIndex],nums[fastIndex]] = [nums[fastIndex],nums[slowIndex]]
            slowIndex++
        }
    }
};

2. 改变矩阵维度

lc

矩阵：m*n 的数字列表 m:代表行 n:代表列 [[1,2],[3,4]],[5,6] 表示3行2列矩阵，js中二维数组表示矩阵

题：给你一个由二维数组 mat 表示的 m x n 矩阵，以及两个正整数 r 和 c ，分别表示想要的重构的矩阵的行数和列数。 重构后的矩阵需要将原始矩阵的所有元素以相同的 行遍历顺序 填充。 如果具有给定参数的 reshape 操作是可行且合理的，则输出新的重塑矩阵；否则，输出原始矩阵。

思考

实际就是使用扁平化化思想

扁平化化思想（二维数组转化为一维数组）：将其中的每个元素 (i,j) 映射到整数域内，并且它们按照行优先的顺序一一对应着 [0,mn),上面都是指的下标，公式为(i,j)→i×n+j,n表示行的长度也就是列，然后从一维转为二维，一维下标x转为矩阵下标i=x / n , j=x % n

适用场景：二维数组转化

解题：直接用长度为m*n的一维数组遍历，二维数组赋值

/**
 * @param {number[][]} mat
 * @param {number} r
 * @param {number} c
 * @return {number[][]}
 */
var matrixReshape = function(mat, r, c) {
    const m = mat.length
    const n = mat[0].length
    
    if(m*n !== r*c){
        return mat
    }

    const arr = new Array(r).fill(0).map(v=> new Array(c).fill(0))
    for(let x=0; x<m*n; x++){
        arr[Math.floor(x/c)][x%c] = mat[Math.floor(x/n)][x%n]
    }
    return arr
};

3.搜索二维矩阵

lc

题：编写一个高效的算法来搜索 m x n 矩阵 matrix 中的一个目标值 target 。该矩阵具有以下特性：

每行的元素从左到右升序排列。每列的元素从上到下升序排列。

思考

1.暴力查找 2.每行有序，可以用二分查找 3.z字形查找

二分查找：取中间，比大小等，等直接返回，大小落区间取中再比，循环条件就是两个边界<=,边界相等就只剩下一个了

场景：有序

/**
 * @param {number[][]} matrix
 * @param {number} target
 * @return {boolean}
 */
var searchMatrix = function(matrix, target) {
    for (const row of matrix) {
        const index = search(row, target)
        if(index >= 0){
            return true
        }
    }
    return false
};

const search = function(nums, target) {
    let start = 0
    let end = nums.length - 1
    while(start <= end) {
        // 注意+start别漏了
        const mid = Math.floor((end - start) / 2) + start
        const curNum = nums[mid]
        if(curNum === target){
            return mid
        }else if(target > curNum) {
            start = mid + 1
        }else {
            end = mid - 1
        }
    }
    return -1
}

z字形查找：以右上角开始搜索，初始(x,y) == (0,n−1), 值相等就ok，value > target 说明这一列都大，那么y减小1，如果value < target说明这一行都小，x增加1。相当于左下角和右上角形成矩阵，超出边界搜索停止

/**
 * @param {number[][]} matrix
 * @param {number} target
 * @return {boolean}
 */
var searchMatrix = function(matrix, target) {
    const m = matrix.length, n = matrix[0].length
    let x = 0, y = n - 1
    // 并集 x
    while(x<m && y>=0) {
        const curValue = matrix[x][y]
        if(curValue === target) {
            return true
        }else if(curValue > target) {
            y--
        }else {
            x++
        }
    }
    return false
};

4.有序矩阵的 Kth Element

lc

题：给你一个 n x n 矩阵 matrix ，其中每行和每列元素均按升序排序，找到矩阵中第 k 小的元素。 请注意，它是 排序后 的第 k 小元素，而不是第 k 个 不同 的元素。 你必须找到一个内存复杂度优于 O(n2) 的解决方案。

思考

1.暴力解决 2.归并排序 3.二分查找

暴力解决：像上面扁平化的思路 整个矩阵放在一位数组里，然后对这个数组排序，最后返回第k值

/**
 * @param {number[][]} matrix
 * @param {number} k
 * @return {number}
 */
var kthSmallest = function(matrix, k) {
    const m = matrix.length
    const n = matrix[0].length
    
    let arr = []
    for(let i = 0; i < m*n; i++){
        arr.push(matrix[Math.floor(i/n)][i%n])
    }
    
    arr.sort((a,b) => a-b)
    return arr[k-1]
};
//s: n^2logn k:O(n^2)

归并排序：归并排序的思想就是将一个数组每个值都分组，然后这些组两两合并，合并时比较组头部有序放入新组，放一次再比较放入，直到有个组为空，剩余组直接放入，然后进行下一对合并。重复操作只剩一组。6->3->2->1

场景： 排序、多个有序组排序

/**
 * @param {number[][]} matrix
 * @param {number} k
 * @return {number}
 */
var kthSmallest = function(matrix, k) {
    const arr = matrix.reduce((a,b) => merge(a,b))
    return arr[k-1]
};

const merge = (left,right) => {
    const n = left.length
    let l = 0, r = 0 , arr = []
    while(l<n && r<n) {
        if(left[l] < right[r]){
            arr.push(left[l++])
        }else {
            arr.push(right[r++])
        }
    }

    while(l < n) {
        arr.push(left[l++])
    }
    while(r < n) {
        arr.push(right[r++])
    }
    return arr
}
// todo 归并+小根堆 s: n^2logn k:O(n^2)

二分查找（前面有算法的讲解）：左上角最小，右下角最大，确定mid（这里mid不是数组下标了就是值），计算多少个数不大于mid（x）,如果x>=k说明第k值<=mid，最大设置为mid,反之最小设置为mid。循环下去最后可以拿到确定值，可以看出这块时间是log(l-r) l是最小值，r是最大值。

其中算出x方法为：从左下脚开始锯齿往右上角，比如当前在左下脚，上面都小，如果比mid大，就往上，反之往右，超出边界结束，时间为O(n)

/**
 * @param {number[][]} matrix
 * @param {number} k
 * @return {number}
 */
var kthSmallest = function(matrix, k) {
    const n = matrix.length
    let left = matrix[0][0]
    let right = matrix[n-1][n-1]
    
    while(left<right){
        const mid =left + Math.floor((right-left)/2)

        if(k <= getCount(matrix,mid,n)){
            right = mid
        }else {
            left = mid + 1
        }
    }
    return left
};

const getCount = (matrix,mid,n) => {
    let i = n - 1, j = 0, count = 0
    while(i>=0 && j<n){
        if(matrix[i][j] <= mid){
            count += (i + 1)
            j++
        }else {
            i--
        }
    }
    return count
}

// s：O(nlog(l-r)) k:O(1)

/**
 * @param {string} s
 * @param {string} t
 * @return {boolean}
 */
var isAnagram = function(s, t) {
    const map = new Map()
    
    for(const i of s) {
        const value = map.get(i) || 0
        map.set(i,value+1)
    }
    for(const i of t) {
        const value = map.get(i) || 0
        if(value){
            if(!(value - 1)){
                map.delete(i)
                continue;
            }
            map.set(i,value - 1)
        }else {
            return false
        }
    }
    if(map.size === 0){
        return true
    }
    return false
};

5.计算一组字符集合可以组成的回文字符串的最大长度

lc

题：给定一个包含大写字母和小写字母的字符串 s ，返回 通过这些字母构造成的最长的回文串的长度。 在构造过程中，请注意 区分大小写 。比如 "Aa" 不能当做一个回文字符串。

思考:

  1. map记录

map记录：排列好的数量都是偶数，奇数最多个一个

字符数map记录 偶数一定能反着排 奇数只有一个能 然后减1累加

或者除以2全部向下取整 ，比较下原长度，不等在+1（偶数都累加，奇数取整累加）

/**
 * @param {string} s
 * @return {number}
 */
var longestPalindrome = function(s) {
    const map = new Map()
    let isHasOne = false
    for(const i of s) {
        const value = map.get(i) || 0
        map.set(i, value+1)
    }

    let count = 0
    for(const item of map) {
        const [_,v] = item
        if(v%2) {
            if(isHasOne){
                count += v-1
            }else{
                count += v
                isHasOne = true
            }
        }else {
            count += v
        }
    }
    return count
};
// s: O(n) k: O(1) 字符表有限，空间不是线性

6.字符串同构

lc

题：给定两个字符串 s 和 t ，判断它们是否是同构的。

如果 s 中的字符可以按某种映射关系替换得到 t ，那么这两个字符串是同构的。

每个出现的字符都应当映射到另一个字符，同时不改变字符的顺序。不同字符不能映射到同一个字符上，相同字符只能映射到同一个字符上，字符可以映射到自己本身。

解：还是创建map

/**
 * @param {string} s
 * @param {string} t
 * @return {boolean}
 */
var isIsomorphic = function(s, t) {
    let obj = {}
    if(s.length!==t.length){
        return false
    }
    for(let i = 0; i < s.length; i++) {
        const isv = s[i]
        const itv = t[i]
        if(!obj[isv]){
            obj[isv] = itv
        }else{
            if(obj[isv] !== itv){
                return false
            }
        }
    }
    return true
};

/**
 * @param {string} s
 * @return {number}
 */
var countSubstrings = function(s) {
    const n = s.length
    let count = 0
    for(let i = 0; i < n; i++) {
        for(let j = i; j < n; j++) {
            // 注意此处substring是左开右闭，所以j要记得+1，不然（0，0）是空字符串

                count++
            }
        }
    }
    return count
};

1. 找出两个链表的交点

lc

给你两个单链表的头节点 headA 和 headB ，请你找出并返回两个单链表相交的起始节点。如果两个链表不存在相交节点，返回 null 。

  1. 双指针解法 2. hash

双指针解法：相交时后面走的路长度一致，那么从两个指针从头开始，如果到了null就到另一个头，还没到null的走到null时就是走了两个链表相差的距离，另一个到达头时后面的距离就是想等的。要么相交相等，要么同时为null结束

/**
 * Definition for singly-linked list.
 * function ListNode(val) {
 *     this.val = val;
 *     this.next = null;
 * }
 */

// 看起来节点可以直接置为null

/**
 * @param {ListNode} headA
 * @param {ListNode} headB
 * @return {ListNode}
 */
var getIntersectionNode = function(headA, headB) {
    if(!headA || !headB) {
        return null
    }
    let pA = headA, pB = headB
    while(pA != pB) {
        pA = pA === null ? headB : pA.next
        pB = pB === null ? headA : pB.next
    }

    return pA
};

2.链表反转

给你单链表的头节点 head ，请你反转链表，并返回反转后的链表。

1.迭代

迭代：两个指针指向前 和 当前，记得最开始前设置null


/**
 * @param {ListNode} head
 * @return {ListNode}
 */
var reverseList = function(head) {
    let prev = null
    let cur = head

    while(cur) {
        let temp = cur.next
        cur.next = prev
        prev = cur
        cur = temp
    }

    return prev
};
// s:O(n) k:O(1) 每次循环定义变量不算奥 下次循环就被释放掉了，额外空间始终占用1

2.递归

递归是一种把执行逻辑单元化的一种方案，每次调用都入栈，开始执行时最上面出栈，依次执行，最后执行到最开始调用。那么入栈得有打断，不然无限入栈，就是函数终止的判断，所以某种角度看这是从后往前执行。

从前往后看的话，那么第一次执行，就是确认后面都是执行过的。

情况很多主要还是以理解，找到可以重复执行的函数输入参数要么是递进的，要在函数里有递进行为参数

上题：单元化我想当前节点和后节点反转，那么后面节点也得反转过，也就是后面都是执行过的，调用的递归，反过来看就是从最后一个节点逐次反转往前

适用场景： 从根往后的结构与孩子节点往后的结构一致，就可以使用递归, 或者是层次化的数据结构，可能出现重复执行的逻辑 都可以用递归

/**
 * Definition for singly-linked list.
 * function ListNode(val, next) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.next = (next===undefined ? null : next)
 * }
 */
/**
 * @param {ListNode} head
 * @return {ListNode}
 */
var reverseList = function(head) {

    if(!head || !head.next) {
        return head
    }
    
    const newHead = reverseList(head.next)
    head.next.next = head
    head.next = null

    return newHead
};

时间复杂度：O(n)，其中 n 是链表的长度。需要对链表的每个节点进行反转操作。

空间复杂度：O(n)，其中 n 是链表的长度。空间复杂度主要取决于递归调用的栈空间，最多为 n 层。

3.归并两个有序的链表

lc

将两个升序链表合并为一个新的 升序 链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。

1.递归 首先找到能够重复执行的点，首先输入是两个头节点，返回新的头节点，实际就是比较输入的两个头节点的大小，，因为是递增的序列 这里出现重复执行点：较小的头节点指向 递归后面都递归执行过的返回的头节点。

/**
 * Definition for singly-linked list.
 * function ListNode(val, next) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.next = (next===undefined ? null : next)
 * }
 */
/**
 * @param {ListNode} list1
 * @param {ListNode} list2
 * @return {ListNode}
 */
var mergeTwoLists = function(list1, list2) {
    if(!list2) {
        return list1
    }else if(!list1) {
        return list2
    }else if(list1.val > list2.val){
        list2.next =  mergeTwoLists(list1, list2.next)
        return list2
    } else if(list1.val <= list2.val) {
        list1.next =  mergeTwoLists(list1.next, list2)
        return list1
    }
};

2.迭代：直接比较迭代

/**
 * Definition for singly-linked list.
 * function ListNode(val, next) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.next = (next===undefined ? null : next)
 * }
 */
/**
 * @param {ListNode} list1
 * @param {ListNode} list2
 * @return {ListNode}
 */
var mergeTwoLists = function(list1, list2) {
    let head = new ListNode(0)
    let prev = head

    while(list1 && list2){
        if(list1.val > list2.val){
            prev.next = list2
            list2 = list2.next
        } else {
            prev.next = list1
            list1 = list1.next
        }
        prev = prev.next
    }

    prev.next = list1 ? list1 : list2

    return head.next
};

4. 从有序链表中删除重复节点

lc

给定一个已排序的链表的头 head ， 删除所有重复的元素，使每个元素只出现一次 。返回 已排序的链表 。

  1. 直接迭代
/**
 * Definition for singly-linked list.
 * function ListNode(val, next) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.next = (next===undefined ? null : next)
 * }
 */
/**
 * @param {ListNode} head
 * @return {ListNode}
 */
var deleteDuplicates = function(head) {
    let cur = head
    while(cur && cur.next) {
        if(cur.val === cur.next.val) {
            const temp = cur.next
            cur.next = cur.next.next
            temp.next = null
        }
        cur = cur.next
    }
    return head
};

2.递归 老样子找重复执行的函数，划分为单元查看一次该执行的逻辑，结尾执行查看如何中断.

/**
 * Definition for singly-linked list.
 * function ListNode(val, next) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.next = (next===undefined ? null : next)
 * }
 */

 * @return {ListNode}

var deleteDuplicates = function(head) {

        return head
    }
    res = deleteDuplicates(head.next)

        return res
    } else {
        head.next = res
        return head
    }
};

5.数组中元素与下一个比它大的元素之间的距离

题：给定一个整数数组 temperatures ，表示每天的温度，返回一个数组 answer ，其中 answer[i] 是指对于第 i 天，下一个更高温度出现在几天后。如果气温在这之后都不会升高，请在该位置用 0 来代替。

单调栈：一个栈只存储递增或者递减的数据

场景：在 O(n) 的时间复杂度内求出数组中各个元素右侧第一个更大的元素及其下标，然后一并得到其他信息。

比如这题右侧第一个温度大的下标，首先第一个数据压入栈，下一个是否大于栈顶，不是继续压入，是的话弹出栈顶并算出下标距离，然后循环往后比较，这样只要在递减的数据都会压入，然后第一次大的一定是最近距离。

这样一直保持一个单调栈

/**
 * @param {number[]} temperatures
 * @return {number[]}
 */
var dailyTemperatures = function(temperatures) {
    let stack = []
    let ans = new Array(temperatures.length).fill(0)
    for(const t in temperatures) {
        while(stack.length && temperatures[t] > temperatures[stack[stack.length - 1]] ) {
            const index = stack.pop()
            ans[index] = t - index 
        }
        stack.push(t)
    }
    return ans
};

时间复杂度：O(n)，其中 n 是温度列表的长度。正向遍历温度列表一遍，对于温度列表中的每个下标，最多有一次进栈和出栈的操作。

空间复杂度：O(n)，其中 n 是温度列表的长度。需要维护一个单调栈存储温度列表中的下标。

1.树的高度

lc

给定一个二叉树 root ，返回其最大深度。

二叉树的 最大深度 是指从根节点到最远叶子节点的最长路径上的节点数。

1.深度优先搜索（递归） 老样子就是往下递归，一个节点连接另外两个树，每个都是，结构相似具有递归特性，找出重复执行的函数（当前高度等于两外两颗树高度最大的+1），max(l,r)+1,这里执行逻辑也有了，找出终止条件，根是null高度为0

/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} root
 * @return {number}
 */
var maxDepth = function(root) {
    if(!root) {
        return 0
    }
    const d = Math.max(maxDepth(root.left),maxDepth(root.right)) + 1
    return d
};

时间复杂度：O(n)，其中 n 为二叉树节点的个数。每个节点在递归中只被遍历一次。

空间复杂度：O(height)，其中 height 表示二叉树的高度。递归函数需要栈空间，而栈空间取决于递归的深度，因此空间复杂度等价于二叉树的高度。

2.广度优先搜索

就是每一层从左往右搜索，逐层往下，需要用一个队列记录当层、下一层的数据

/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} root
 * @return {number}
 */
var maxDepth = function(root) {
    const queue = [root]
    let depth = 0
    while(queue.length>0) {
        let size = queue.length
        while(size>0) {
            const i = queue.shift()
            const il = i.left
            const ir = i.right
            if(il) {
                queue.push(il)
            }
            if(ir) {
                queue.push(ir)
            }
            size--
        }
        depth++
    }

    return depth
};

时间复杂度：O(n)，其中 n 为二叉树的节点个数。与方法一同样的分析，每个节点只会被访问一次。

空间复杂度：此方法空间的消耗取决于队列存储的元素数量，其在最坏情况下会达到 O(n)。

6.判断路径和是否等于一个数

给你二叉树的根节点 root 和一个表示目标和的整数 targetSum 。判断该树中是否存在 根节点到叶子节点 的路径，这条路径上所有节点值相加等于目标和 targetSum 。如果存在，返回 true ；否则，返回 false 。

1.广度优先，每一层遍历时，同时记录每一层的路径值，与广度遍历同步，如果是叶子节点就去判断当前路径是否符合目标

/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} root
 * @param {number} targetSum
 * @return {boolean}
 */
var hasPathSum = function(root, targetSum) {
    if(!root) {
        return false
    }
    const q1 = [root]
    const q2 = [root.val]

    while(q1.length > 0){
        let size = q1.length
        while(size > 0) {
            const node = q1.shift()
            const val = q2.shift()
            const left = node.left
            const right = node.right
            if(left) {
                q1.push(left)
                q2.push(val + left.val)
            }
            if(right) {
                q1.push(right)
                q2.push(val + right.val)
            }

            if(!left && !right && val === targetSum) {
                return true
            }
            size --
        }
    }
    return false
};

时间复杂度：O(N)，其中 N 是树的节点数。对每个节点访问一次。

空间复杂度：O(N)，其中 N 是树的节点数。空间复杂度主要取决于队列的开销，队列中的元素个数不会超过树的节点数。

 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {

/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} root
 * @param {number} low
 * @param {number} high
 * @return {TreeNode}
 */
var trimBST = function(root, low, high) {
    if(!root) {
        return null
    }

    const left = trimBST(root.left, low, high)
    const right = trimBST(root.right, low, high)

    if(root.val < low) {
        return right
    }
    if(root.val > high) {
        return left
    }

    root.left = left
    root.right = right

    return root
};

var quicksort = (nums, l, r) => {
    if(l>=r) {
        return
    }
    const pos = partition(nums, l, r)

    quicksort(nums, l, pos - 1)
    quicksort(nums, pos + 1, r)
}

var partition = (nums, l, r) => {
    var randomi = Math.floor(Math.random() * (r + 1 - l) + l)
    [nums[randomi], nums[r]] = [nums[r], nums[randomi]]
    const pivot = nums[r]
    let i = l - 1, j = l
    while(j < r) {
        if(nums[j] > pivot) {
            j++
        }else {
            i++
            [nums[i],nums[j]] = [nums[j],nums[i]]
            j++
        }
    }
    [nums[i+1], nums[r]] = [nums[r], nums[i+1]]
    return i + 1
}

    let p = i

        const l = p * 2 + 1, r = p * 2 + 2

归并排序

归并排序的思想就是将一个数组每个值都分组，然后这些组两两合并，合并时比较组头部有序放入新组，放一次再比较放入，直到有个组为空，剩余组直接放入，然后进行下一对合并。重复操作只剩一组。6->3->2->1

const merge = (nums,l,r) => {
    if(l >= r) {
        return [nums[l]]
    }

    const center = Math.floor((r - l)/2 + l)
    const left = merge(nums, l, center)
    const right = merge(nums, center + 1, r)

    let i = 0, j = 0, temp = []
    while(i < left.length && j < right.length) {
        if(left[i] > right[j]) {
            temp.push(right[j++])
        }else {
            temp.push(left[i++])
        }
    }
    
    while(i < left.length) {
        temp.push(left[i++])
    }
    while(j < right.length) {
        temp.push(right[j++])
    }

    return temp
    
}
